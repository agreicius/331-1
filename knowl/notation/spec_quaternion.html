Specimen 9. Quaternion group.

Let \(Q_8=\{1,-1,i,-i,j,-j,k,-k\}\text{.}\) The following rules define a binary operation \(\cdot\) on \(Q_8\text{.}\)
\begin{align*} 1\cdot a \amp =a\cdot 1=a \text{ for all } a\in Q_8\\ (-1)\cdot (-1) \amp =1 \amp (-1)\cdot a\amp=a\cdot (-1)=-a \text{ for all } a\in Q_8\\ i^2 \amp =j^2=k^2=-1\\ ij \amp =k \amp ji\amp =-k\\ jk \amp = i \amp kj\amp =-i\\ ki \amp = j \amp ik\amp =-j\text{.} \end{align*}
The pair \((Q_8,\cdot)\) is a group called the quaternion group. We have the following presentation of \(Q_8\text{:}\)
\begin{equation*} Q_8=\angvec{i,j\mid i^2=j^2=(ij)^2, ji=i^{-1}j}\text{.} \end{equation*}
