Specimen 26. Quaternion rings.

Let \(R\) be a nontrivial commutative ring. We define \(H(R)\) to be the set of all formal sums
\begin{gather*} a_1+a_2i+a_3j+a_4k\text{,} \end{gather*}
where \(a_i\in R\) for all \(1\leq i\leq 4\text{.}\) We define addition on \(H(R)\) as
\begin{gather*} (a_1+a_2i+a_3j+a_4k)+(b_1+b_2i+b_3j+b_4k) \\ = (a_1+b_1)+(a_2+b_2)i+(a_3+b_3)j+(a_4+b_4)k\text{.} \end{gather*}
We define multiplication to be the unique associative binary operation \(\cdot\) that distributes over the addition operation \(+\text{,}\) satisfies
\begin{align} b(a_1+a_2i+a_3j+a_4k) \amp =(a_1+a_2i+a_3j+a_4k)b \tag{2.15}\\ \amp =ba_1+ba_2i+ba_3j+ba_4k\tag{2.16} \end{align}
for all
\begin{align*} b \amp =b+0i+0j+0k\in H(R)\text{,} \end{align*}
and satisfies
\begin{align} i^2\amp =j^2=k^2=-1 \tag{2.17}\\ ij \amp = k, jk=i, ki=j\text{.}\tag{2.18} \end{align}
The triple \((H(R), +, \cdot)\) is a ring called the quaternion ring over (or with coefficients in) \(R\).
