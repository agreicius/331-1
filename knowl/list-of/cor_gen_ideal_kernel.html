Corollary 2.49. Generated ideals and kernels.

Let \(\phi\colon R\rightarrow S\) be a ring homomorphism. For any set \(A\subseteq R\text{,}\) we have \((A)\subseteq \ker\phi\) if and only if \(\phi(a)=0\) for all \(a\in A\text{.}\)
