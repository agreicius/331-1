Corollary 2.61. Quotient ring homomorphisms.

Let \(I\) be an ideal of the ring \(R\text{,}\) and let \(\pi\colon R\rightarrow R/I\) be the quotient map. Given any ring \(S\text{,}\) the map
\begin{align} \Hom(R/I,S) \amp\rightarrow \Hom(R,S)\tag{2.49}\\ \overline{\phi} \amp \mapsto \overline{\phi}\circ \pi\tag{2.50} \end{align}
is an injection onto the set of all \(\phi\in \Hom(R,S)\) satisfying \(I\subseteq \ker \phi\text{.}\) In other words, we can identify \(\Hom(R/I, S)\) with the set
\begin{align*} \{\phi\in \Hom(R,S)\mid I\subseteq \ker\phi\} \amp \text{.} \end{align*}
