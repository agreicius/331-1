Theorem 2.15.1. Polynomial UFDs.

Let \(R\) be a nonzero commutative ring. For all positive integers \(n\text{,}\) \(R[x_1,x_2,\dots, x_n]\) is a unique factorization domain if and only if \(R\) is a unique factorization domain.

Proof.

The proof is by induction on \(n\text{.}\)

Base case: \(n=1\text{.}\) TheoremÂ 2.14.6 implies that if \(R\) is a UFD, then so is \(R[x_1]\text{.}\) Conversely, if \(R[x_1]\) is a UFD, it is easy to see that \(R\) is a UFD as well: using a degree argument, irreducible factorizations of an element \(c\in R\) corresponds bijectively to irreducible factorizations of the constant polynomial \(f(x)=c\) in \(R[x]\text{.}\)

Induction step: let \(n> 1\text{,}\) and assume the result is true for \(n-1\text{.}\) We have
