Theorem 2.101. CRT: integers.

Let \(a_1,a_2,\dots, a_n\) be pairwise relatively prime integers (i.e., \(\gcd(a_i, a_j)=1\) for all \(i\ne j\)), and let \(a=\prod_{k=1}^na_i\text{.}\)
  1. The map
    \begin{align*} \phi\colon \Z/a\Z \amp \rightarrow \prod_{k=1}^n \Z/a_k\Z\\ [m]_a \amp \mapsto ([m]_{a_1}, [m]_{a_2},\dots, [m]_{a_n}) \end{align*}
    is an isomorphism of rings.

  3. For any choice of integers \(m_1, m_2, \dots, m_n\text{,}\) the set of \(x\) satisfying the system of congruences
    \begin{align*} x \amp \equiv m_1 \pmod{a_1}\\ x \amp \equiv m_2 \pmod{a_2}\\ \amp \vdots \\ x \amp \equiv m_n \pmod{a_n} \end{align*}
    forms a unique congruence class modulo \(a\text{.}\) That is, (i) there is a solution \(x\in \Z\) to this system of congruences, and (ii) \(y\) is another solution if and only if \(x\equiv y\pmod a\text{.}\)
