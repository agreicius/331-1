Proposition 1.190. Homomorphisms from cyclic groups.

Let \(G=\angvec{g}\) be a cyclic group, and let \(H\) be a group.
  1. If \(\ord g=\infty\text{,}\) then the map
    \begin{align*} \Hom(G,H) \amp \rightarrow H\\ \phi \amp \mapsto \phi(g) \end{align*}
    is a bijection of sets. Equivalently, for all \(\phi\in \Hom(G,H)\) there is a unique \(h\in H\) such that
    \begin{equation*} \phi(g^i)=h^i \end{equation*}
    for all \(i\in \Z\)

Proof.

  1. Since any \(\phi\in \Hom(G,H)\) satisfies
    \begin{equation*} \phi(g^i)=(\phi(g))^i\text{,} \end{equation*}
    it is clear that a homomophism \(\phi\) is uniquely determined by \(h=\phi(g)\text{.}\) This means that the map
    \begin{equation*} \phi\mapsto \phi(g) \end{equation*}
    is an injection. To see that it is a surjection, we claim that for any \(h\in H\text{,}\) the map \(\phi\colon G\rightarrow H\) defined as
    \begin{equation*} \phi(g^i)=h^i \end{equation*}
    is a group homomorphism satisfying \(\phi(g)=h\text{.}\) As usual, the only real work that needs to be done here is to show that the map map is well defined; and this follows since \(g^i=g^j\) if and only if \(i=j\text{.}\)

  2. The proof is similar. Again, it is clear that a homomorphism \(\phi\in \Hom(G,H)\) is uniquely determined by \(h=\phi(g)\text{;}\) furthermore, since \(g^n=e\text{,}\) we must have
    \begin{equation*} h^n=(\phi(g))^n=\phi(g^n)=\phi(e)=e\text{.} \end{equation*}
    This proves that the map
    \begin{equation*} \phi\mapsto \phi(g) \end{equation*}
    is an injection from \(G\) to \(\{h\in H\mid h^n=e\}\text{.}\) Once again that this map is surjective follows from the fact that given any \(h\in H\) the map \(\phi\colon G\rightarrow H\) defined as
    \begin{equation*} \phi(g^i)=h^i \end{equation*}
    is a homomorphism. In this case, the fact that this map is well defined follows from the fact that in \(G\) we have
    \begin{equation*} g^i=g^j \iff i\equiv j\pmod n\text{.} \end{equation*}
